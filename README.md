
Open [http://localhost:3000](http://localhost:3000) to view the showcase.

## Deployment

This showcase is ready to deploy on Vercel:

1. Push to GitHub
2. Import project in Vercel
3. Deploy automatically

Or use Vercel CLI:

```bash
npm i -g vercel
vercel
```

## Structure

- `app/page.tsx` - Main showcase page
- `components/ui/` - UI components from the library
- `lib/` - Utility functions
